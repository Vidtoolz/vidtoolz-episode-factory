Shorts Workflow v1.0.0

Vertical Short · 9:16 · 1080×1920 · up to 3 min. Blunt, funny, direct monologue.

1 Topic

Pick an existing topic or type a new one. No idea-triage or long-form research needed.

Open Topic Scout

2 Choose from 3 scripts

Generate three short monologue scripts (local Ollama), pick one, edit it, and save as final-script.md.

Script commitment check

Before generating images and video, check whether this script is worth spending media-generation effort on. Advisory only — you stay in control.

3 Image prompts + images

Generate FLUX image prompts, then images — or copy each text-to-image prompt (Copy buttons in the Image Prompts editor) to generate manually in GPT and upload the result below. Generated images are viewable in the tools below.

Image Prompts Generate / view images Select images
Add a manual (GPT) image:

4 Image-to-video prompts

Generate I2V motion prompts from your script (local Ollama). Copy each individually, or save them to the package's video-prompts.json for PRESTO.

5 Send to PRESTO

Send images + I2V prompts to PRESTO for image-to-video generation. Existing PRESTO safety boundaries are preserved.

Open PRESTO controls

6 View generated videos

Review the generated videos in the GUI.

AIGEN Review

7 Record + edit in Resolve (manual)

Vertical workflow ends here. Record your monologue to camera, bring the generated videos and your recording into DaVinci Resolve, edit manually, and publish the Short. No Resolve automation or publish-ready status is forced for this path.
