    New video project

    Step 1

    Title

    Generate fills the box with a local-Ollama (vidnux) topic — click Save to keep it.
    Step 2

    Script / voiceover

    Generate writes a local-Ollama (vidnux) script from the saved title — click Save to keep it.
    Step 3

    Main image prompts

    Create image prompts replaces the whole set (asks first). Create remaining prompts fills only the empty slots — up to 100 — and leaves your existing prompts, images, and video prompts untouched. All 100 slots stay editable; edit any box, then Copy or Save changes.
    ⚠ The script changed after these prompts were generated. They may be stale — regenerate, or keep your edits.
    Step 4

    Generated images

    Images to generate is an upper bound. The batch only targets rows that have a prompt and no image yet (rows with an existing image or an empty prompt are skipped), in row order — so entering more than are eligible just generates the eligible ones. Run again to continue where it left off. One batch at a time (vidnux ComfyUI / FLUX); images appear next to each prompt above as they finish. ComfyUI must be running.
    Step 5

    Infographic prompts

    Prompt count is an upper bound. This fills only empty infographic slots (up to 30) from the saved script — existing prompts are preserved, never replaced. Run again to add more; clear a row to free its slot. Edit any box, then Copy / Save / Clear.
    ⚠ The script changed after these prompts were generated. They may be stale — regenerate, or keep your edits.
    Step 6

    Image-to-video prompts

    Fills an image-to-video prompt only for generated images that don't have one yet (PRESTO Ollama lane). Existing i2v prompts are never overwritten. Each image row above also has its own Create / Regenerate / Clear i2v prompt controls.
    Step 7

    Generated videos

    Runs PRESTO ComfyUI / Wan2.2 on rows that have BOTH a generated still and a saved image-to-video prompt. One clip at a time, slow (minutes each), resumes and skips finished clips. PRESTO must be running; it is never auto-started. Each clip appears next to its image above; or use the per-image “Generate video on PRESTO” button.
